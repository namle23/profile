3D Configurator (On going project)

React-3D configurator from JSON

During summer 2018, I'm working on a project using React/Redux and ThreeJS to build a dynamica 3D configurator from JSON. Each model has many instances and will be shown when users click on each model. Instances will be displayed in 3D format as well. This configurator let user imports their own JSON file (this JSON file has to follow a certain format) and the configurator will display those 3D model of products (including prices, name, code,...) automatically. Users also can modify the components from the original product (display in 3D format) according to their need, on the web.

Happy chef

An e-commerce site developed with HTML, CSS, Java EE, JSP with EL and JSTL, about a bakery store. Customer can choose among many kind of cake, add to cart and checkout. When order, the customer who logged in don't need to enter information as it's loaded from database. All the orders will be added to processing status in database.
